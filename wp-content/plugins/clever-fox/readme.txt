== Description ==

Clever Fox plugin to enhance the functionality of free themes made by <a href="https://nayrathemes.com/" rel="nofollow">Nayra Themes</a>. More than 60000+ trusted websites with Nayra Themes. It provides intuitive features to your website. 35+ Themes compatible with Clever Fox. See below free themes listed here. <a href="https://wordpress.org/themes/avril/" rel="nofollow">Avril</a>, <a href="https://wordpress.org/themes/gradiant/" rel="nofollow">Gradiant</a>, <a href="https://wordpress.org/themes/flavita/" rel="nofollow">Flavita</a>, <a href="https://wordpress.org/themes/conceptly/" rel="nofollow">Conceptly</a> & <a href="https://wordpress.org/themes/colorpress/" rel="nofollow">ColorPress</a> is one of highest installations themes in our collections. Install any theme via single click install and import demo instantly. 

**Features:**

- Import Pre-made Section in 1 Click
- Beautiful Sections on Homepage
- Edit content via Live Customizer
- Manage Font Typography
- Background Animations
- Kenburn Effect on Slider
- Compatible with WPML & Polylang
- Translation Ready
- Responsive for All Devices
- Cross Browser Compatibility
- and More Features

== Installation ==

= Option 1. =

- Upload the entire `clever-fox` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Clever Fox" —> Click install.
- After plugin is downloaded, click active plugin
